Rimtify is a modern, full-stack web application that streamlines university operations by providing specialized dashboards for administrators, teachers, and students. Built with Next.js, Supabase, and Framer Motion, it delivers a seamless, real-time experience for managing academic workflows.

https://preeti-kaur245.github.io/RIMTify-saas/

Key Features
------------For Administrators---------
Academic Structure Management

Department Hierarchy Builder: Create and manage departments, programs, semesters, and sections with an interactive tree view
Curriculum Designer: Map subjects to specific sections and semesters with drag-and-drop allocation
Faculty Assignment Dashboard: Assign teachers to subjects and sections with automatic conflict detection
Student Enrollment Manager: Manual and bulk enrollment with CSV import support

Smart Attendance System

GPS-Based Verification: Configurable radius-based attendance with real-time location tracking
QR Code Support: Alternative attendance marking method
Security Levels: Low, Medium, High security modes with anti-cheat measures
Session Duration Control: Customizable attendance windows (30s-2min)

Financial Management

Fee Tracking: Assign, monitor, and track student fee payments
Payroll System: Manage teacher salaries with bonus and deduction tracking
Revenue Dashboard: Real-time pending fee calculations and financial overview

Data Operations

Bulk Import Center: CSV-based import for students, teachers, subjects, and enrollments
Preview System: Validate and review data before committing to database
Promotion Wizard: Batch-promote students from one semester to the next

Library & Resources

Book Management: Track library inventory with loan status
Catalog System: Digital book registration with author and ISBN tracking

Communication Hub

Broadcast Announcements: Role-based messaging (students, teachers, all)
Priority Levels: General, urgent, and event-based notifications


-----------For Teachers---------
Intelligent Attendance

Dual-Mode Attendance:

Manual Mode: Traditional roster-based marking with search and filters
Smart Mode: GPS + temporary code generation for automated marking


Real-Time Classroom Feed: Live view of students marking attendance
Session History: Access past attendance records with one-click restore
Attendance Analytics: Class-wide performance metrics and trend analysis

Class Management

My Classes Dashboard: Visual cards showing all assigned programs, semesters, sections, and subjects
Quick Class Switching: Dropdown selector for managing multiple classes
Student Roster Import: CSV upload for bulk student addition to sections

Grade Management

Multi-Exam System: Internal, midterm, and final exam grade entry
Bulk Grade Submission: Save grades for entire class in one operation
Performance Tracking: Individual student grade history

Course Materials

Resource Upload: Share PDFs, notes, and external links
Subject Organization: Materials categorized by course
Direct File Access: Students download materials instantly

Student Analytics

Performance Dashboard: Top 3 and bottom 3 performers
Individual Profiles: Detailed attendance and grade history per student
Class Distribution: Visual progress bars showing student attendance percentages

Payroll Tracking

Salary Slip Access: View monthly payment history
Breakdown Details: Base salary, bonuses, deductions, and net pay


--------For Students-------
Personal Dashboard

Attendance Overview: Real-time attendance percentage with visual progress
Smart Alerts: Notifications for absent markings
Enrollment Status: Display of assigned program, semester, and section

Smart Attendance Join

Code Entry System: Enter teacher-generated temporary codes
GPS Verification: Automatic location validation (must be within classroom radius)
Real-Time Feedback: Instant confirmation or rejection with distance display
Session Timeout: Time-sensitive entry windows for security

Academic Records

Exam Results: View all grades (internal, midterm, final) with scores
Course Materials: Download lecture notes, PDFs, and resources
Subject Access: Materials filtered by enrolled courses

Financial Management

Fee Dashboard: Track pending and paid fees with due dates
Payment History: Complete transaction records
Amount Breakdown: Fee title, amount, status, and payment date

Library Integration

Issued Books: View currently borrowed books
Due Dates: Track return deadlines
Author & ISBN Info: Complete book metadata

Communication

Announcements Feed: Receive campus-wide and role-specific notifications
Priority Filtering: Urgent alerts highlighted

--------Technology Stack---------
Frontend

Next.js 15.1 - React framework with App Router
TypeScript - Type-safe development
Framer Motion - Smooth animations and transitions
Lucide React - Modern icon library
Custom CSS - Glass morphism and neon design system

Backend

Supabase - PostgreSQL database with real-time subscriptions
Row Level Security (RLS) - Role-based access control
Authentication - Email/password with JWT tokens
Real-time Listeners - Live attendance feed updates

------Key Features--------

Responsive Design - Mobile-first with desktop optimization
Progressive Web App (PWA) - Installable on devices
Real-time Sync - Instant updates across all clients
Geolocation API - GPS-based attendance verification
File Uploads - Cloud storage integration ready
